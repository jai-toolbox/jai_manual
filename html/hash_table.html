Hash Table

A hash table is a data structure that maps keys to values. It works by feeding each key through a hash function to produce an integer, which is then used to compute an index into an internal array of slots, allowing lookups, insertions, and removals in approximately constant time. The type of a hash table mapping keys of type K to values of type V is written Table(K, V), and lives in the "Hash_Table" module.

Note: We will not go over all the details, but a simple high level usage here. For the details go read the code for it. 

    
#import "Hash_Table";

scores: Table(string, int);
table_add(*scores, "Alice", 95);
table_add(*scores, "Bob",   82);

value, success := table_find(*scores, "Alice");
if success  print("Alice's score is %\n", value);  // 95

Adding and Setting Entries

Use table_add to insert a key-value pair. It returns a pointer to the inserted value. If you add the same key twice, both entries will exist and lookups will return the first match. Use table_set instead if you want at most one entry per key; it replaces the value if the key already exists. 

    
table_add(*scores, "Carol", 91);       // Always inserts a new entry.
table_set(*scores, "Alice", 97);       // Replaces Alice's existing score.
table_set(*scores, "Dave",  88);       // Adds Dave, since he wasn't in the table.

Looking Up Values

Use table_find to look up a key. It returns both the value and a success boolean. The success return is marked #must, so the compiler will error if you discard it. If success is false, the returned value is uninitialized and must not be used. Use table_find_pointer if you need a pointer to the value, for example to modify it in place. It returns null if the key is not found. 

    
value, success := table_find(*scores, "Alice");
if success  print("Alice's score is %\n", value);

ptr := table_find_pointer(*scores, "Bob");
if ptr  ptr.* += 10;   // Give Bob 10 extra points.

Checking for a Key

Use table_contains when you only need to know whether a key exists. This is useful when using a table as a set: 

    
if table_contains(*scores, "Carol") {
    print("Carol is in the table.\n");
}

Removing Entries

Use table_remove to remove the first entry matching a given key. It returns a success boolean and the value that was removed: 

    
success, removed_value := table_remove(*scores, "Bob");
if success  print("Removed Bob with score %\n", removed_value);

Find or Add

Use find_or_add when you want a pointer to a value for a given key, creating a default-initialized entry if the key does not yet exist: 

    
entry, newly_added := find_or_add(*scores, "Eve");
if newly_added  entry.* = 75;

Iteration

Hash tables can be iterated with a for loop. The loop variable it holds the value, and it_index holds the key. The iteration order is not guaranteed. 

    
for scores {
    print("% has score %\n", it_index, it);
}

You can iterate by pointer to modify values in place, and you can remove entries during iteration: 

    
for * scores {
    it.* += 5;   // Give everyone 5 bonus points.
}

for scores {
    if it < 60  remove it;   // Remove all failing scores.
}

Resetting and Freeing

Use table_reset to clear all entries while keeping the allocated memory. Use deinit to free all memory owned by the table: 

    
table_reset(*scores);   // count is now 0, memory still allocated.
deinit(*scores);        // All memory freed.
